# Aplikasi Plastik Keselamatan SPM

Fail utama:
- index.html
- style.css
- script.js
- data.js

Cara guna di GitHub/Netlify:
1. Upload semua fail ini ke repository.
2. Pastikan `index.html` berada di root repository.
3. Deploy melalui GitHub Pages atau Netlify.

Jika mahu hantar ke Google Sheets:
1. Deploy Google Apps Script sebagai Web App.
2. Salin URL `/exec`.
3. Buka `script.js`.
4. Masukkan URL pada bahagian `const SCRIPT_URL = '';`.
